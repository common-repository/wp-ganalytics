== Description ==

The plugin will help you to add Google Analytics code in your website to track user's different actions.

### Features of Google Analytics Plugin:

* **User friendly and easy to use.**
* **Tested with other popular plugins.**
* **Fully Customizable.**
* **Ability to change Analytics Property ID.**

### If you found any bug in our Google Analytics Plugin or have a question contact us at milonfci@gmail.com.

== Installation ==

### That's simple:

1. Download from Wordpress.org Google Analytics Plugin.
2. Upload the Google Analytics plugin into plugins directory "wp-content/plugins"
3. Activate the Google Analytics plugin from admin panel.
4. Go to Settings->Google Analytics and update your Google Analytics Tracking No.
